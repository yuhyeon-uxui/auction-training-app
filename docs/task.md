## v2 확장 계획 (종잣돈 모인 후)

### 핵심 방향
v1이 "배우는 앱"이라면, v2는 "실전에서 쓰는 앱".
종잣돈이 생겼을 때 바로 실전 투입할 수 있도록
공부하면서 익힌 감각을 실제 분석 도구로 연결한다.

### 추가 예정 기능

#### 실전 경매 분석 도구
- 실제 대법원 경매 물건 링크 붙여넣기 → 자동 정보 파싱
- DSR/LTV 기반 대출 가능액 계산기
- 실거래가 vs 감정가 비교 (KB시세 연동)
- 권리분석 체크리스트 (인수 금액 자동 계산)
- n년 후 시세 예측 (보수적/중립/낙관적 3가지)

#### 기술 스택 확장
- FastAPI 백엔드 추가
  (현재 프론트에서 처리하는 계산 로직을 서버로 이관)
- 실제 경매 데이터 크롤링 or 외부 API 연동
- 구글 OAuth 실연동 (현재 UI만 구현된 상태)
- 애플 로그인 (Apple Developer 계정 필요, 연 $99)

#### 커뮤니티 기능
- 내 입찰 분석 공유하기
- 다른 유저 입찰가 비교 (익명)
