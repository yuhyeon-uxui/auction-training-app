# 🎨 디자인 가이드 (Design Guide)

본 문서는 '부동산 경매 가상 입찰 훈련소'의 일관된 사용자 경험(UX)과 시각적 디자인(UI) 원칙을 정의합니다.

## 1. 글로벌 테마 및 색상 (Theme & Colors)

전체적인 분위기는 최신 트렌드의 **SaaS 대시보드 형태의 다크모드**를 지향하며, 눈의 피로를 덜어주는 어두운 배경에 블루를 포인트 컬러로 사용하여 신뢰감과 세련미를 줍니다.

- **기본 배경 (Background)**: 어두운 차콜/블랙 계열 (`#0a0a0a`, `#111111`)
- **포인트 컬러 (Accent Color)**: 
  - 기본 상태: 밝은 블루 `#3B82F6` (Tailwind 기본 파란색 기반)
  - Hover/Active 상태: 진한 블루 `#2563EB`
- **텍스트 컬러 (Text)**: 
  - 기본 본문: 밝은 회색/흰색 계열 (`#f0f0f0`)
  - 보조 텍스트(Muted): 어두운 회색 (`#a0a0a0`)
- **버튼 텍스트**: 완전한 흰색 `#FFFFFF`

## 2. 레이아웃 구조 (Layout)

데스크탑 및 태블릿 환경에 최적화된 고정형 뷰포트 구조를 가집니다.

- **좌측 고정 사이드바 (Sidebar)**: 앱 로고 및 메인 네비게이션(홈, 개념 배움터, 훈련소, 프로필) 고정 배치
- **상단 헤더 (Header)**: 현재 페이지 타이틀 및 알림/요약 정보
- **메인 콘텐츠 (Main Content)**: 우측 영역에 그리드(Grid) 시스템을 활용하여 카드 컴포넌트들을 넓고 쾌적하게 배치

## 3. UI 컴포넌트 스타일링 원칙 (Style Rules)

딱딱한 금융 서비스가 아닌 **트렌디하고 부드러운 에듀테크 앱**의 느낌을 주도록 스타일링합니다.

- **라운드 코너 (Rounded Corners)**: 모든 버튼과 카드 배경 모서리에 적절한 둥글기(`border-radius`)를 적용하여 부드러운 인상을 줍니다.
- **얇은 보더 라인 (Thin Borders)**: 얇고 미세한 어두운 회색 보더를 사용하여 칠흑 같은 배경 속에서도 구역을 깔끔하고 세련되게 구분합니다.
- **부드러운 트랜지션 (Transitions)**: 버튼에 마우스를 올리거나(Hover) 카드를 조작할 때, 색상이 변하거나 살짝 떠오르는 애니메이션을 부드럽게 적용하여 앱이 '살아 숨 쉬는' 듯한 동적인 인터랙션을 제공합니다.
- **글로우 효과 (Glow/Shadow)**: 프로그레스 바나 핵심 CTA(콜투액션) 버튼에는 블루 컬러의 희미한 그림자(box-shadow)를 주어 네온처럼 은은하게 빛나는 효과를 줍니다.
